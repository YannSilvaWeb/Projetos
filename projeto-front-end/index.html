O que você irá aprender

Aprenda a desenvolver interfaces de alta qualidade através de uma metodologia simples e com exemplos práticos.

Desenvolvimento HTML3 + CSS3 + JS

Três tecnologias essenciais para o seu aprendizado de front-end, já que elas são a base para a codificação de qualquer layout.

Desenvolvimento Wordpress

Será abordado como funciona uma estrutura de Wordpress na criação de um site gerenciável. Em seguida, criaremos juntos um tema com blog e suas funcionalidades essenciais.

Desenvolvimento React JS

Iremos desenvolver um portfólio que poderá ser utilizado posteriormente como apresentação dos seus trabalhos. Além de criarmos o html, o projeto será desenvolvido em React usando o NextJs.

Evolua com o que há de melhor nomercado

Domine as principais técnicas para se tornar um front-end capaz de codificar qualquer tipo de layout.
